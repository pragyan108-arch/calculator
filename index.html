📘 Calculator Help Guide

0–9 → Enter numbers

+ − × ÷ → Addition, Subtraction, Multiplication, Division

= → Show result

C → Clear screen

← → Delete last character

. → Decimal point

x² → Square of number

xʸ → Power function

√x → Square root

1/x → Reciprocal

( ) → Parentheses grouping

sin, cos, tan → Trigonometric functions

DEG/RAD → Switch angle mode

log → Base 10 logarithm

ln → Natural logarithm

Enter → Calculate

Backspace → Delete last character

Escape → Clear screen

Numbers & operators can be typed directly

History panel stores previous calculations.

If expression is invalid, calculator shows Error.

Check parentheses and operators if error appears.
